## 🚀 Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Basic understanding of HTML, CSS, and JavaScript

### Installation

1. **Clone or download the repository:**
   ```bash
   git clone <repository-url>
   cd "my portifolio"
   ```

2. **Open in your browser:**
   - Simply open `index.html` in your web browser
   - Or serve using your preferred local server (Python, Node.js, etc.)

3. **For local server (Python):**
   ```bash
   # Python 3
   python -m http.server 8000
   
   # Then navigate to http://localhost:8000
   ```

## 📱 Responsive Breakpoints

The design uses Bootstrap's responsive grid system with mobile-first approach:
- **Extra Small (xs):** < 576px
- **Small (sm):** ≥ 576px
- **Medium (md):** ≥ 768px
- **Large (lg):** ≥ 992px
- **Extra Large (xl):** ≥ 1200px
- **XXL (xxl):** ≥ 1400px

## 🔧 Troubleshooting

### Images not loading?
- Check file paths in HTML are relative to project root
- Ensure image files exist in the `images/` directory
- Verify file names match exactly (case-sensitive)

### Styles not applying?
- Clear browser cache (Ctrl+Shift+Del or Cmd+Shift+Del)
- Check CSS file paths are correct
- Inspect element in DevTools to debug

### Navigation highlighting not working?
- Ensure sections have unique `id` attributes
- Check that `script.js` is loaded correctly
- Open browser console for any JavaScript errors
